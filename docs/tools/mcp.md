# Model Context Protocol (MCP)

Model Context Protocol (MCP) 是 Antigravity 與外部數據源及工具連接的標準化方式。

## Core Features (核心特色)

*   **Connectivity**: 允許 Agent 與廣泛的服務、API 和資料庫進行無縫互動。
*   **Extensibility**: 透過實作 MCP Servers，使用者可以擴充 Agent 的能力，包含自定義動作與數據存取模式。
*   **Protocol-Based**: 確保 Agent 與異質系統溝通時具有一致的介面，使其成為複雜整合任務的可靠解決方案。

透過 MCP，Agent 可以超越單純的對話，轉變為能夠操作真實世界工具與調用數據的強大助手。
