# 模型上下文協定 (MCP)

模型上下文協定 (Model Context Protocol, MCP) 是 Antigravity 與外部數據源及工具連接的標準化方式。

## 核心特色

*   **連通性 (Connectivity)**：允許代理與廣泛的服務、API 和資料庫進行無縫互動。
*   **擴充性 (Extensibility)**：透過實作 MCP 伺服器，使用者可以擴充代理的能力，包含自定義動作與數據存取模式。
*   **基於協定 (Protocol-Based)**：確保代理與異質系統溝通時具有一致的介面，使其成為複雜整合任務的可靠解決方案。

透過 MCP，代理可以超越單純的對話，轉變為能夠操作真實世界工具與調用數據的強大助手。
