# 工具概述 (Tools Overview)

Google Antigravity 提供了一套整合工具來增強 AI 代理的能力，讓它能夠與外部系統互動、管理複雜任務，並利用專案特定的資訊。

## 主要工具類別

*   **模型上下文協定 (Model Context Protocol, MCP)**：與外部數據源與工具連接的標準方式。
*   **產出物 (Artifacts)**：在工作流程中建立與更新的結構化、持久化文件。
*   **知識庫 (Knowledge)**：索引專案特定的文檔與上下文資訊，讓代理具備深度洞察力。

這些工具共同協作，讓代理在中大型專案開發中能夠更加精確地執行任務並維持高品質的產出。
