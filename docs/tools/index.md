# Tools Overview (工具概述)

Google Antigravity 提供了一套整合 Tools 來增強 Agent 的能力，讓它能夠與外部系統互動、管理複雜任務，並利用專案特定的資訊。

## Key Tool Categories (主要工具類別)

*   **Model Context Protocol (MCP)**：與外部數據源與工具連接的標準方式。
*   **Artifacts**：在 Workflow 中建立與更新的結構化、持久化文件。
*   **Knowledge**：索引專案特定的文檔與上下文資訊，讓 Agent 具備深度洞察力。

這些工具共同協作，讓 Agent 在中大型專案開發中能夠更加精確地執行任務並維持高品質的產出。
