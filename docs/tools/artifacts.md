# 產出物

**產出物** (Artifacts) 是代理執行任務時產生的可驗證文件，讓您能夠追蹤進度並審核代理的工作。

## 產出物類型

### 📋 任務清單 (task.md)

追蹤任務進度的待辦事項清單。

```markdown
# 任務

## 階段一：設置
- [x] 創建專案結構
- [x] 初始化 Git
- [/] 安裝相依套件

## 階段二：實作
- [ ] 實作登入功能
- [ ] 撰寫單元測試
```

**標記說明**：
- `[ ]` 未完成
- `[/]` 進行中
- `[x]` 已完成

### 📝 實作計畫 (implementation_plan.md)

在規劃模式產生，詳細說明將進行的變更。

**內容包含**：
- 目標描述
- 需要用戶審核的項目
- 檔案變更清單
- 驗證計畫

### 📖 操作指南 (walkthrough.md)

在驗證模式產生，記錄已完成的工作。

**內容包含**：
- 變更摘要
- 測試結果
- 截圖與錄影

### 📸 截圖與錄影

視覺化的執行證明，嵌入在產出物中。

## 產出物位置

產出物儲存在專案根目錄的 `.gemini/antigravity/brain/` 資料夾中。

## 與產出物互動

- 點擊產出物連結可在編輯器中開啟
- 可以直接編輯產出物內容
- 代理會根據您的修改更新計畫

---

> **提示**：保持任務清單更新有助於代理理解目前進度。
