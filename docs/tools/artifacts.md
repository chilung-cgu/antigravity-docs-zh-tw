# 產出物 (Artifacts)

產出物是在代理工作流程中建立與更新的結構化、持久性文件。它們記錄了代理的意圖、進度與最終結果。

## 主要產出物類型

### 1. 任務清單 (Task List)
*   **用途**：追蹤專案中所有子任務的狀態。
*   **功能**：清晰呈現已完成、進行中與後續待辦事項的總覽。

### 2. 實作計畫 (Implementation Plan)
*   **用途**：架構完成任務所需的技術變更。
*   **功能**：在修改程式碼庫前由代理生成，供使用者審查，確保提議的架構符合預期。

### 3. 操作指南 (Walkthrough)
*   **用途**：解釋已完成的工作。
*   **功能**：任務完成後提供變更細節，幫助使用者理解修改內容及其影響。

### 4. 截圖 (Screenshots)
*   **用途**：提供視覺驗證。
*   **功能**：代理在執行期間擷取瀏覽器或應用程式狀態，對 UI/UX 除錯與驗證至關重要。

### 5. 瀏覽器錄影 (Browser Recordings)
*   **用途**：提供詳細的逐步視覺日誌。
*   **功能**：自動錄製 WebP 影片，擷取代理與瀏覽器的互動過程，允許使用者回放並審核操作細項。
