# Knowledge (知識庫)

Knowledge 功能讓 Agent 能夠利用特定專案獨有的文檔與上下文資訊。

## Key Features (關鍵特色)

*   **Project Context**: 使用者可以索引相關文件（如風格指南、API 文檔或內部 README），讓 Agent 深入理解專案。
*   **Efficient Retrieval**: Agent 可以搜尋並從 Knowledge 提取資訊，以做出明智決策並遵循本地編碼標準。
*   **Continuous Learning**: 隨著專案文檔演進，Knowledge 可隨時更新，確保 Agent 與最新需求保持一致。

![Knowledge View Pane](../assets/images/knowledge-pane.png)

藉由將專案知識內化，Agent 能產出更精確且符合團隊規範的程式碼與建議。
