# 知識庫 (Knowledge)

知識庫功能讓代理能夠利用特定專案獨有的文檔與上下文資訊。

## 關鍵特色

*   **專案上下文 (Project Context)**：使用者可以索引相關文件（如風格指南、API 文檔或內部 README），讓代理深入理解專案。
*   **高效檢索 (Efficient Retrieval)**：代理可以搜尋並從知識庫提取資訊，以做出明智決策並遵循本地編碼標準。
*   **持續學習 (Continuous Learning)**：隨著專案文檔演進，知識庫可隨時更新，確保代理與最新需求保持一致。

藉由將專案知識內化，代理能產出更精確且符合團隊規範的程式碼與建議。
