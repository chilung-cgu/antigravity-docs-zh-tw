# 代理管理器概述 (Agent Manager Overview)

代理管理器 (Agent Manager) 提供了一個更高層級的視圖，讓您在您的引導下監督 Antigravity 代理所進行的工作。

## 核心價值

*   **跨工作空間協作**：同時管理多個工作空間與數十個代理。
*   **以代理為中心**：主要透過代理與程式碼庫互動，而非直接編寫程式碼。
*   **鳥瞰視圖**：這將成為所有未來工作的主要入口。

## 如何切換

*   **快捷鍵**：按下 **CMD+E** (Mac) 或 **CTRL+E** (Windows/Linux) 即可在編輯器與代理管理器之間切換。
*   **切換按鈕**：使用選單欄右上角的「開啟編輯器 (Open Editor)」或「開啟代理管理器 (Open Agent Manager)」按鈕。

> [!NOTE]
> 代理管理器目前處於公開預覽 (Public Preview) 階段，我們正在持續迭代並將其定位為核心體驗。
