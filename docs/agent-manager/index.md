# 代理管理器

**代理管理器** (Agent Manager) 是 Antigravity 的任務控制中心，讓您能夠管理多個工作空間和監控代理活動。

## 概覽

代理管理器提供以下功能：

| 功能 | 說明 |
|------|------|
| **工作空間** | 管理多個專案 |
| **遊樂場** | 實驗與測試環境 |
| **收件匣** | 代理通知與訊息 |
| **對話視圖** | 對話歷史記錄 |
| **瀏覽器子代理** | 瀏覽器任務監控 |

## 開啟代理管理器

您可以透過以下方式開啟：

1. 選單列：`視圖 > 代理管理器`
2. 快捷鍵：`Cmd+Shift+M` (macOS) / `Ctrl+Shift+M` (Windows/Linux)
3. 命令面板：`開啟代理管理器`

## 工作空間管理

在代理管理器中，您可以：

- 建立新工作空間
- 切換工作空間
- 設定工作空間偏好
- 查看工作空間統計
