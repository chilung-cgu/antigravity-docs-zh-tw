# Agent Manager Overview (代理管理器概述)

Agent Manager 提供了一個更高層級的視圖，讓您在您的引導下監督 Antigravity Agent 所進行的工作。

## Core Values (核心價值)

*   **Cross-Workspace Collaboration**: 同時管理多個 Workspaces 與數十個 Agents。
*   **Agent-Centric**: 主要透過 Agent 與 Codebase 互動，而非直接編寫程式碼。
*   **Bird's Eye View**: 這將成為所有未來工作的主要入口。

## How to Switch (如何切換)

*   **Shortcut**: 按下 **CMD+E** (Mac) 或 **CTRL+E** (Windows/Linux) 即可在 Editor 與 Agent Manager 之間切換。
*   **Toggle Button**: 使用選單欄右上角的「Open Editor」或「Open Agent Manager」按鈕。

> [!NOTE]
