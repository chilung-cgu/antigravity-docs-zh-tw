# Inbox (收件匣)

Inbox 是您追蹤所有對話的一站式中心。

## Functions (功能介紹)

在 Inbox 中，您可以集中管理所有需要您介入的對話，包括：
*   **Permission Requests**: 核准 Terminal 指令的執行。
*   **Browser Requests**: 授權 Agent 使用 Browser 獲取資訊。
*   **Plan Review**: 審閱並核准 Agent 生成的 Implementation Plan。

透過 Inbox，您可以高效地管理多個並行任務的進度。
