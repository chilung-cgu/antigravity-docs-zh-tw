# 收件匣 (Inbox)

收件匣是您追蹤所有對話的一站式中心。

## 功能介紹

在收件匣中，您可以集中管理所有需要您介入的對話，包括：
*   **權限請求**：核准終端機指令的執行。
*   **瀏覽器請求**：授權代理使用瀏覽器獲取資訊。
*   **計畫審查**：審閱並核准代理生成的實作計畫。

透過收件匣，您可以高效地管理多個並行任務的進度。
