# 對話視圖 (Conversation View)

對話視圖中的**代理面板 (Agent Panel)** 是代理管理器的核心。

## 核心體驗

*   **即時追蹤**：隨著代理取得進展，您可以即時觀察其操作步驟。
*   **跟隨模式 (Following Mode)**：預設情況下，視圖會自動跟隨代理的操作。您可以點擊右上角的「正在跟隨 (Following)」按鈕來關閉此模式，以便自由瀏覽對話紀錄。
