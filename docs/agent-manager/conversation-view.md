# Conversation View (對話視圖)

Conversation View 中的 **Agent Panel** 是 Agent Manager 的核心。

## Core Experience (核心體驗)

*   **Real-time Tracking**: 隨著 Agent 取得進展，您可以即時觀察其操作步驟。
*   **Following Mode**: 預設情況下，視圖會自動跟隨 Agent 的操作。您可以點擊右上角的「Following」按鈕來關閉此模式，以便自由瀏覽對話紀錄。
