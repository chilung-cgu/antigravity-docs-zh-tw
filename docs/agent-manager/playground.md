# 遊樂場 (Playground)

遊樂場是獨立的工作空間，讓您能夠立即開始對話並探索想法，而無需進行繁瑣的專案設定。

## 建立遊樂場

1.  進入「開始對話 (Start Conversation)」頁面。
2.  在輸入框下方點擊「使用遊樂場 (Use Playground)」按鈕。
3.  直接發送訊息開始探索。

遊樂場非常適合用於快速測試程式碼片段、詢問技術問題或進行腦力激盪。
