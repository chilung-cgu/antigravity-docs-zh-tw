# Playground (遊樂場)

Playground 是獨立的 Workspace，讓您能夠立即開始對話並探索想法，而無需進行繁瑣的專案設定。

## Creating a Playground (建立遊樂場)

1.  進入「Start Conversation」頁面。
2.  在輸入框下方點擊「Use Playground」按鈕。
3.  直接發送訊息開始探索。

Playground 非常適合用於快速測試程式碼片段、詢問技術問題或進行腦力激盪。
