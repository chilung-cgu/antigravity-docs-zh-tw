# Workspaces (工作空間)

在 Agent Manager 中，您可以同時啟動並切換多個 Workspaces。

## Operations (操作說明)

*   **Open Workspace**: 點擊左側邊欄的按鈕，選擇一個起始資料夾。
*   **Multitasking**: 您可以隨時透過左側邊欄在不同 Workspaces 的對話之間切換，無需關閉目前的任務。

這讓您能夠在處理多個專案或任務時，依然維持清晰的組織結構。
