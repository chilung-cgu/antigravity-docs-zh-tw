# 工作空間 (Workspaces)

在代理管理器中，您可以同時啟動並切換多個工作空間。

## 操作說明

*   **開啟工作空間**：點擊左側邊欄的按鈕，選擇一個起始資料夾。
*   **多工切換**：您可以隨時透過左側邊欄在不同工作空間的對話之間切換，無需關閉目前的任務。

這讓您能夠在處理多個專案或任務時，依然維持清晰的組織結構。
