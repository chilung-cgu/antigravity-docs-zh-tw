# Antigravity 術語解釋表 (Terminology Glossary)

> **注意**：本專案採用「保留英文專有名詞」策略。本表僅供了解術語的中文含義與解釋使用，**請勿**在文檔中將英文術語替換為中文。

## Core Concepts (核心概念)

| English Term | Explanation (中文解釋) |
| :--- | :--- |
| **Agent** | 代理。Antigravity 的核心 AI 助手，能執行規劃、編碼、驗證等任務。 |
| **Antigravity** | Google 開發的新一代 AI IDE。 |
| **Model** | 模型。驅動 Agent 的 AI 模型 (如 Gemini 3 Pro)。 |
| **Space** | 空間。 |
| **User** | 使用者。 |

## Interface (介面)

| English Term | Explanation (中文解釋) |
| :--- | :--- |
| **Editor** | 編輯器。編寫程式碼的主要介面。 |
| **Agent Manager** | 代理管理器。管理 Agent、工作空間與任務的介面。 |
| **Agent Side Panel** | 代理側邊欄。編輯器右側與 Agent 互動的面板。 |
| **Conversation View** | 對話視圖。顯示 Agent 執行進度與對話紀錄的介面。 |
| **Playground** | 遊樂場。用於快速實驗與對話的臨時空間。 |
| **Inbox** | 收件匣。集中管理等待審核的請求與通知。 |
| **Browser** | 瀏覽器。Antigravity 內建或控制的 Chrome 瀏覽器。 |
| **Terminal** | 終端機。 |
| **Tab** | 分頁 / Tab 鍵。常用於程式碼補全的操作。 |

## Functional Terms (功能術語)

| English Term | Explanation (中文解釋) |
| :--- | :--- |
| **Artifacts** | 產出物。Agent 生成的結構化文件（如計畫、截圖、錄影）。 |
| **Workflows** | 工作流程。預定義的自動化任務腳本。 |
| **Rules** | 規則。引導 Agent 行為的指令文件。 |
| **Settings** | 設定。 |
| **Modes** | 模式。Agent 的運作模式（如 Planning, Execution）。 |
| **Implementation Plan** | 實作計畫。Agent 在寫 code 前產生的技術規劃文檔。 |
| **Walkthrough** | 操作指南 / 導覽。任務完成後的總結報告。 |
| **Task List** | 任務清單。 |
| **Knowledge** | 知識庫。專案特定的背景資料索引。 |
| **Review Changes** | 審查變更。提交代碼前的審核介面。 |
| **Source Control** | 原始碼控制 (Git)。 |
| **Supercomplete** | 超級補全。編輯器中的 AI 智慧補全功能。 |

## Tools & Technical (工具與技術)

| English Term | Explanation (中文解釋) |
| :--- | :--- |
| **MCP (Model Context Protocol)** | 模型上下文協定。連接外部工具的標準協定。 |
| **Chrome Extension** | Chrome 擴充功能。 |
| **Allowlist** | 允許清單。 |
| **Denylist** | 拒絕清單。 |
| **Profile** | 設定檔 (瀏覽器)。 |
