# 貢獻指南

感謝您對本翻譯專案的興趣！以下是參與貢獻的指南。

## 翻譯規範

### 文風

- **語調**：專業但親切，避免過於生硬
- **人稱**：使用「您」作為第二人稱
- **語態**：盡量使用主動語態

### 標點符號

使用全形標點符號：

| 類型 | 正確 | 錯誤 |
|------|------|------|
| 逗號 | ， | , |
| 句號 | 。 | . |
| 冒號 | ： | : |
| 分號 | ； | ; |
| 問號 | ？ | ? |
| 驚嘆號 | ！ | ! |
| 引號 | 「」 | "" |
| 括號 | （） | () |

### 代碼處理

- 代碼區塊內容保持英文原文
- 代碼註解可視情況翻譯
- 使用三個反引號並標註程式語言

```javascript
// 這是範例代碼
const greeting = "Hello, World!";
```

### 術語一致性

- 請參考 [GLOSSARY.md](GLOSSARY.md) 中的術語對照表
- 新術語需先在 Issue 中討論後再加入術語表

## 提交流程

1. Fork 本專案
2. 建立功能分支：`git checkout -b translate/agent-models`
3. 進行翻譯並提交
4. 發起 Pull Request

## 檔案命名

- 使用小寫英文與連字號
- 範例：`getting-started.md`、`chrome-extension.md`

## 品質檢查

提交前請確認：

- [ ] 無錯別字
- [ ] 術語與術語表一致
- [ ] Markdown 語法正確
- [ ] 內部連結有效

---

如有問題，歡迎開 Issue 討論！
