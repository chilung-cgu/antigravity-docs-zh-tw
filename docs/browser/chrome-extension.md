# Chrome 擴充功能 (Chrome Extension)

Antigravity Chrome 擴充功能是代理存取網頁的必要元件。

## 功能與作用

*   **視覺化觀察**：讓代理能夠「看見」並與網站元素互動。
*   **互動增強**：允許使用者直接從瀏覽器端執行取消對話等操作。

## 安裝指南

正常情況下，第一次啟動瀏覽器代理時，系統會引導您至 Chrome 線上應用程式商店安裝。

### 手動安裝步驟
如果需要手動安裝：
1.  點擊 Antigravity 中的 **Chrome 圖示**（這會以隔離的設定檔啟動瀏覽器）。
2.  在編輯器右上角或代理管理器左下角找到圖示。
3.  導航至擴充功能網址，點擊「加到 Chrome」。
