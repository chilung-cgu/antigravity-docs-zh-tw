# Chrome 設定檔 (Chrome Profiles)

為了確保安全與隔離，Antigravity 會在獨立的 Chrome 設定檔中運作。

## 隔離優點

*   **資訊不共享**：不會分享您平常使用的 Cookie 或登入狀態。
*   **持久性**：在 Antigravity 設定檔中進行的登入資訊會被保留，方便後續工作。
*   **無干擾**：Antigravity 擴充功能不會影響您平常的瀏覽器。

## 如何自定義

如果您希望更改設定檔的儲存位置，可以在設定中修改 **「Browser User Profile Path」**。
