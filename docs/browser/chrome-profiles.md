# Chrome Profiles (Chrome 設定檔)

為了確保安全與隔離，Antigravity 會在獨立的 Chrome Profile 中運作。

## Isolation Benefits (隔離優點)

*   **Data Privacy**: 不會分享您平常使用的 Cookie 或登入狀態。
*   **Persistence**: 在 Antigravity Profile 中進行的登入資訊會被保留，方便後續工作。
*   **No Interference**: Antigravity Extension 不會影響您平常的 Browser。

## Customization (如何自定義)

如果您希望更改 Profile 的儲存位置，可以在 Settings 中修改 **「Browser User Profile Path」**。
