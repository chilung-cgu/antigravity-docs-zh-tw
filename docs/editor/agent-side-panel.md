# 代理側邊欄 (Agent Side Panel)

代理側邊欄位於編輯器右側，是您與 AI 代理互動的主要介面。

## 主要功能

*   **對話管理**：啟動新對話、附加圖片背景資訊，以及切換**代理模式**。
*   **模型選擇**：在不同的模型（如 Gemini 3 Pro, Gemini 3 Flash）之間切換。
*   **追蹤工具列**：輸入框上方的工具列會追蹤已開啟的檔案變更、執行中的終端機進程以及活耀的產出物 (Artifacts)。
