# Commands (行內指令)

對於較小的任務與快速修改，**Command** 功能允許您直接在 Editor 行內請求補全或執行 Terminal 指令。

## Usage (使用方式)

*   **Trigger Shortcut**: `Cmd + I` (Mac) 或 `Ctrl + I` (Windows/Linux)。
*   **Workflow**: 游標處會出現文字輸入框，您可以用自然語言輸入指令（例如：「建立一個排序清單的函式」）。
*   **Execution**: Antigravity 會直接行內生成程式碼或建議 Terminal 指令，供您核准或修改。
