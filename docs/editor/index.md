# Editor Overview (編輯器概述)

Antigravity Editor 是您使用平台進行開發的主要入口。它基於 **VS Code** Codebase 構建，提供熟悉的開發體驗，同時增強了量的 AI 功能。

## Core Features (核心特點)

*   **Familiar Interface**: 支援開啟檔案、Tab 管理以及直接編輯，與 VS Code 操作一致。
*   **Extension Support**: 相容 Open VSX 市集，可安裝語法高亮、版本控制整合等套件。
*   **Deep AI Integration**: 旨在與 Agent 無縫協作，無論是小型任務還是大型專案。
