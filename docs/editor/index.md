# 編輯器

Antigravity 建構在現代化的程式碼編輯器之上，提供熟悉的開發體驗，同時深度整合 AI 代理功能。

## 介面配置

```
┌─────────────────────────────────────────────┐
│  選單列 | 分頁列                             │
├───────────┬─────────────────────┬───────────┤
│           │                     │           │
│  檔案     │                     │  代理     │
│  總管     │     編輯區域         │  側邊欄   │
│           │                     │           │
├───────────┴─────────────────────┴───────────┤
│  面板區域（終端機 / 輸出 / 問題）             │
└─────────────────────────────────────────────┘
```

## 核心功能

| 功能 | 說明 |
|------|------|
| **分頁** | 多檔案分頁管理 |
| **命令** | 快捷命令面板 |
| **代理側邊欄** | 與代理互動 |
| **審查變更** | 代碼審查工具 |
| **原始碼控制** | Git 整合 |

## 快捷鍵

| 動作 | macOS | Windows/Linux |
|------|-------|---------------|
| 命令面板 | `Cmd+Shift+P` | `Ctrl+Shift+P` |
| 快速開啟檔案 | `Cmd+P` | `Ctrl+P` |
| 開啟代理側邊欄 | `Cmd+Shift+A` | `Ctrl+Shift+A` |
| 儲存檔案 | `Cmd+S` | `Ctrl+S` |
