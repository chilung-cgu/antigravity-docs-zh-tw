# 編輯器概述 (Editor Overview)

Antigravity 編輯器是您使用平台進行開發的主要入口。它基於 **VS Code 程式碼庫**構建，提供熟悉的開發體驗，同時增強了豐富的 AI 功能。

## 核心特點

*   **熟悉的介面**：支援開啟檔案、分頁管理以及直接編輯，與 VS Code 操作一致。
*   **擴充功能支援**：相容 Open VSX 市集，可安裝語法高亮、版本控制整合等套件。
*   **AI 深度整合**：旨在與 AI 代理無縫協作，無論是小型任務還是大型專案。
