# Source Control (原始碼控制)

Editor 支援標準的 Source Control 整合，讓您在審核完 Agent 的工作後，可以輕鬆地與您的版本控制系統連動。

## Key Features (主要特色)

*   **Git Integration**: 完整支援常見的 Git 操作。
*   **Changes Sidebar**: 提供詳細的變更列表與管理功能。
*   **Workflow Compatibility**: 您可以先完成 Agent 的變更審核，再進行標準的原始碼提交。
