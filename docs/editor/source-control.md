# 原始碼控制 (Source Control)

編輯器支援標準的原始碼控制整合，讓您在審核完代理的工作後，可以輕鬆地與您的版本控制系統連動。

## 主要特色

*   **Git 整合**：完整支援常見的 Git 操作。
*   **變更側邊欄 (Changes Sidebar)**：提供詳細的變更列表與管理功能。
*   **工作流程相容**：您可以先完成 AI 代理的變更審核，再進行標準的原始碼提交。
