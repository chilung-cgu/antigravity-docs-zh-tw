# 智能代理 (Agent)

智能代理是 Google Antigravity 的核心 AI 功能。它是一個由尖端大語言模型 (Frontier LLM) 驅動的多步驟推理系統。其關鍵能力包括：
*   對現有程式碼庫進行推理。
*   利用廣泛的工具，包括內建瀏覽器。
*   透過任務、產出物及其他互動元素與用戶溝通。

## 核心組件
智能代理的架構由以下部分組成：
*   **推理模型 (Reasoning model)**：提供智慧的底層大語言模型 (LLM)。
*   **工具 (Tools)**：代理可以呼叫的功能功能擴充。
*   **產出物 (Artifacts)**：代理建立的結果輸出或結構化數據。
*   **知識庫 (Knowledge)**：代理可取用的上下文資訊與數據。

## 自定義設定
用戶可以透過以下方式定制代理體驗：
*   **代理模式 / 設定 (Agent Modes / Settings)**：調整操作參數。
*   **MCP (Model Context Protocol)**：用於擴充上下文或工具能力。
*   **規則 / 工作流程 (Rules / Workflows)**：為代理定義標準化的操作程序。

## 其他資訊
*   **多代理支援**：用戶可以同時進行多個代理對話。
*   **管理**：代理對話可以透過代理管理器（右鍵點擊 > 刪除對話）或透過編輯器代理面板中的垃圾桶圖示進行刪除。
