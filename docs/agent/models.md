# 模型 (Models)

Antigravity 支援多種高性能的大語言模型，您可以根據任務需求選擇適合的模型。模型選擇在單一對話中具有「黏性」(Sticky)，即會保持您的選擇。

## 推理模型 (Reasoning Models)

使用者可以從以下推理模型中選擇：

*   **Gemini 3 Pro Preview**：最強大的預覽版模型。
*   **Gemini 2.5 Pro**：穩定且強大的推理模型。
*   **Claude Sonnet 4.5**：擅長程式碼生成與邏輯推理。
*   **Claude Sonnet 4.5 (Thinking)**：具備更深層思考能力的專用版。
*   **GPT-OSS 120B (Medium)**：高性能的開源替代方案。

## 專用模型 (Specialized Models)

這些模型用於特定的後端任務，使用者無法自定義：

*   **Nano Banana Pro**：生成式繪圖工具，用於製作 UI 模型、填寫網頁內容，以及生成系統或架構圖。
*   **Gemini 2.5 Pro UI Checkpoint**：輔助瀏覽器子代理 (Browser subagent) 的互動與 UI 處理。
*   **Gemini 2.5 Flash**：用於內部檢查點 (Internal checkpointing) 與上下文摘要。
*   **Gemini 2.5 Flash Lite**：驅動程式碼庫內的語義搜尋。
