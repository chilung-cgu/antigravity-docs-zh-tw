## 🌐 關於 Antigravity

Google Antigravity 是 Google 於 2025 年 11 月發布的新一代 AI 驅動整合開發環境 (IDE)，採用「代理優先」設計理念，讓 AI 代理能夠自主規劃、執行、驗證軟體開發任務。

### 主要特色

- **代理優先架構** - AI 代理可跨編輯器、終端、瀏覽器執行任務
- **產出物與驗證** - 生成任務清單、實作計畫、截圖等供用戶驗證
- **多介面整合** - 編輯器、瀏覽器、代理管理器無縫協作
- **多模型支援** - Gemini 3 Pro、Claude Sonnet 4.5、GPT-OSS 等
