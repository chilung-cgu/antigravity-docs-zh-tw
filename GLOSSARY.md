# 術語表 (Glossary)

本術語表統一專案中使用的翻譯術語，確保一致性。

## 核心概念

| English | 繁體中文 | 說明 |
|---------|----------|------|
| Agent | 代理 | AI 智能代理，可自主執行任務 |
| Artifact | 產出物 | 代理工作產出的可驗證檔案 |
| Workflow | 工作流程 | 自定義的自動化任務流程 |
| IDE | 整合開發環境 | Integrated Development Environment |
| MCP | MCP | Model Context Protocol，保留英文 |

## 使用者介面

| English | 繁體中文 | 說明 |
|---------|----------|------|
| Agent Manager | 代理管理器 | 管理多個代理的控制台 |
| Agent Side Panel | 代理側邊欄 | 編輯器中的代理互動面板 |
| Workspace | 工作空間 | 專案工作區域 |
| Playground | 遊樂場 | 實驗與測試環境 |
| Inbox | 收件匣 | 代理訊息通知區 |
| Conversation View | 對話視圖 | 與代理的對話記錄 |
| Side Panel | 側邊欄 | 可摺疊的側邊面板 |
| Tabs | 分頁 | 編輯器分頁標籤 |
